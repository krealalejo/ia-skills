# 🧠 IA Skills Repository

Una colección de habilidades (skills) personalizadas para agentes de IA, diseñadas para potenciar flujos de trabajo de ingeniería de software, automatización y mejores prácticas de desarrollo.

## 🚀 Skills Incluidas

### 🛠️ git-commit
**Propósito**: Automatiza la creación de commits atómicos siguiendo el estándar de [Conventional Commits](https://www.conventionalcommits.org/).

**Características principales**:
- **Mensajes en Inglés**: Forzado de idioma para consistencia en repositorios profesionales.
- **Commits Atómicos**: Agrupa cambios lógicamente por módulo o funcionalidad.
- **Soporte de Tickets**: Integra IDs de tickets (ej. Jira, GitHub) automáticamente en el scope.
- **Límite de Caracteres**: Garantiza mensajes concisos (máx. 300 caracteres).

## 🛠️ Cómo usar este repositorio

Este repositorio está diseñado para ser clonado o referenciado por agentes de IA compatibles con el sistema de **Antigravity Skills**.

1. **Instalación**: Copia la carpeta de la skill deseada a tu directorio de skills local.
2. **Configuración**: Asegúrate de que el agente tenga los permisos necesarios para ejecutar los comandos definidos en el archivo `SKILL.md`.
